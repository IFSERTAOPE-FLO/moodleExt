# 📖 Glossário de Termos — MoodleExt:

Este documento define os termos técnicos e educacionais utilizados no contexto do projeto MoodleExt para garantir uma linguagem comum entre todos os colaboradores.

## 🎓 Termos do Moodle (Core):

| Termo | Descrição |
| :--- | :--- |
| **Ambiente Virtual (AVA)** | O sistema como um todo, onde ocorrem as atividades de ensino e aprendizagem. |
| **Bloco (Block)** | Itens que podem ser adicionados às laterais das páginas (ex: Calendário, Avisos). |
| **Curso (Course)** | A unidade básica de conteúdo, onde professores postam materiais e alunos realizam atividades. |
| **Inscrição (Enrolment)** | O processo de adicionar um usuário (aluno/professor) a um curso específico. |
| **Papel (Role)** | Define as permissões de um usuário no sistema (ex: Aluno, Professor, Tutor, Administrador). |
| **Plugin** | Extensão de software que adiciona uma funcionalidade específica ao Moodle. |
| **Recurso vs Atividade** | **Recurso** é algo para ler/ver (PDF, link). **Atividade** é algo para fazer (Fórum, Questionário). |

---

## 🚀 Termos Específicos do MoodleExt:

| Termo | Descrição |
| :--- | :--- |
| **Sertão Conecta** | Nome da identidade visual e do conceito pedagógico central do projeto. |
| **Gamificação** | Conjunto de mecânicas (XP, Medalhas, Níveis) integradas para aumentar o engajamento. |
| **Component Library** | Biblioteca customizada de elementos de interface (UI) exclusivos do projeto. |
| **Raiz Pública Segregada** | Modelo de segurança que separa a lógica do sistema (`moodleExt/`) dos arquivos acessíveis via web (`public/`). |

---

## 🏗️ Termos Técnicos (Desenvolvimento):

| Termo | Descrição |
| :--- | :--- |
| **AMD (Asynchronous Module Definition)** | Padrão utilizado para carregar módulos JavaScript no Moodle. |
| **Grunt** | Executor de tarefas utilizado para compilar SCSS e processar JavaScript. |
| **Mustache** | Sistema de templates lógica-less utilizado para renderizar o HTML da interface. |
| **SCSS (Sass)** | Pré-processador de CSS que permite o uso de variáveis e funções para estilos mais organizados. |
