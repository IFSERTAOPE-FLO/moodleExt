# Tela de Login

Ponto de acesso seguro para usuários cadastrados na plataforma Sertão Conecta.

## 1. Elementos da Interface

- **Formulário Central**:
- Campo de Usuário (E-mail institucional ou CPF).
- Campo de Senha.
- Opção "Lembrar identificação".
- **Ações**:
- Botão "Acessar".
- Link "Esqueceu sua senha?".
- Botão de "Criar nova conta" (redireciona para Cadastro).
- **Identidade**:
- Uso das cores Verde e Branco.
- Logotipo do projeto em destaque.

## 2. Requisitos Funcionais:

- Validação de campos obrigatórios.
- Mensagens de erro claras para credenciais inválidas.
- Suporte a autenticação via Moodle (LDAP ou banco de dados local).
